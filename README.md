# Ascian Music Player - Umbra Plugin

A custom plugin for Umbra that allows you to add a Widget to be used with [Ascian Music Player](https://github.com/GayPotatoEmma/Ascian-Music-Player) (available on Dalamud Main Repo)
> [!IMPORTANT]
> **Requires Ascian Music Player v1.2.1.2 or later!**

## Features
- Media Controls
- Interactive seekbar
- Album art support

## How to install
- **Automatic (Recommended)**
  - Open Umbra Settings
  - Navigate to Plugins
  - Set the owner and repository
    - Owner: `GayPotatoEmma`
    - Repository: `Umbra.AscianMusicPlayer`
  - Click Add Repository
  - Confirm and Restart Umbra

- **Manual**
  - Grab the [latest release](https://github.com/GayPotatoEmma/Umbra.AscianMusicPlayer/releases/latest) and extract it into a folder of your choosing
  - Open Umbra Settings
  - Navigate to Plugins
  - Press the Browse... button on the Install from File section
  - Navigate to the extracted folder and choose the `.dll`
  - Confirm and Restart Umbra
